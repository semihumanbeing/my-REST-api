# myRestAPI
project to make a REST api server

## Requirement
A small RESTful api controller that has get/post/put/delete endpoints<br>
It's using Mybatis mapper from controller to the Oracle database<br>

## Stack
Java 1.8<br>
Spring Boot<br>
Spring Web<br>
Maven<br>
Oracle database<br>
ojdbc 8<br>
Tomcat<br>
MyBatis 2.2.2<br>
